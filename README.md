## Run Locally

Clone the project

```bash
  git clone https://github.com/sndvny/trainer_laravel10.git
```

Go to the project directory

```bash
  cd trainer_laravel10
```

Install dependencies

```bash
  composer Install
```

Optimize

```bash
  php artisan optimize:clear
```

Generate app key

```bash
  php artisan key:generate
```

Migrasi database

```bash
  php artisan migrate
```

Run

```bash
php artisan serve
```
